## Reporting a Vulnerability

If you should spot a vulerability, please contact @GreyXC with the following information:

- What release and branch the vulnerability is found
- A brief description & what type of vulnerability
- Suggestions on how to fix it (if it hasn't been fixed already)

  From there a discussion on next steps will be taken
